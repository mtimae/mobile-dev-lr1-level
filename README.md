# ЛР1 — Цифровой уровень (горизонтали)

## Описание проекта
Мобильное Android-приложение «Цифровой уровень», разработанное в рамках лабораторной работы по дисциплине «Разработка мобильных приложений».

Приложение использует акселерометр смартфона для определения отклонения устройства от горизонтального положения и отображает:
- положение виртуального пузырька;
- значения наклона по осям X и Y;
- текстовый статус положения устройства;
- виброотклик при почти горизонтальном положении.

## Тема лабораторной работы
Работа с сенсорами

## Вариант
Вариант 2 — «Цифровой уровень (горизонтали)»

## Используемые технологии
- Kotlin
- Android Studio
- XML Layout
- SensorManager
- SensorEventListener
- Vibrator / VibrationEffect

## Функциональные возможности
- считывание данных акселерометра;
- отображение наклона по осям X и Y;
- визуальное перемещение пузырька внутри рабочей области;
- определение почти горизонтального положения;
- короткая вибрация при выравнивании;
- корректная регистрация и отмена регистрации слушателя датчика.

## Структура проекта
- `MainActivity.kt` — основная логика приложения;
- `activity_main.xml` — интерфейс приложения;
- `level_frame.xml` — рамка рабочей области;
- `bubble_circle.xml` — пузырёк уровня;
- `AndroidManifest.xml` — конфигурация приложения и разрешение на вибрацию.

## Инструкция по запуску
1. Установить Android Studio.
2. Открыть проект.
3. Подключить Android-смартфон по USB.
4. Включить режим разработчика и USB-отладку на телефоне.
5. Выбрать устройство в Android Studio.
6. Нажать `Run`.

## Инструкция по использованию
1. Запустить приложение на смартфоне.
2. Положить телефон на поверхность или плавно наклонять его.
3. Наблюдать за перемещением пузырька.
4. Смотреть изменения значений X и Y.
5. При почти горизонтальном положении проверить изменение статуса и вибрацию.

## Результат
Приложение успешно определяет наклон устройства по данным акселерометра и визуально отображает положение смартфона относительно горизонтали.
